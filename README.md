# lexia

Lenguaje de programación para crear leyes.

## ¿Por qué?

Porque hay un orden lógico en el lenguaje y también puede explotarse en el ámbito de las leyes.

## ¿Para qué?

Algún día, tu juez será una máquina programada por el humano.

## Uso

Expongo aquí algunos ejemplos, y luego voy con la sintaxis.

### Ejemplo 1

```
================================
=== REGLAS DEL COMEDOR COMÚN ===
================================

ARTÍCULO 1: Obligación de limpiar todo rastro propio

SE OBLIGA A limpiar todo rastro de mierda
  A todo usuario del comedor
  CON PENA DE multa de +0.20€
  CON ATENUANTE de -0.10€
  	SI tenía una urgencia

ARTÍCULO 2: Prohibición de entrar con zapatos

SE PROHIBE entrar en el comedor
  A todo usuario del comedor
  CON PENA DE multa de +0.40€
  CON AGRAVANTE de +0.25€
  	SI pisó una mierda de perro

```
